# 🤝 Contributing to fullstack-saas-stack

Thank you for your interest! This repository is part of the **GitHub Open Source Knowledge Restructuring Project**.

## How to Contribute

### 1. Add a New Project Analysis
Create `projects/[project-name].md` following the TEMC template (see ai-agent-stack for examples).

### 2. Update comparison.md and SOURCES.md

### 3. Extract Universal Code
If you find cross-category patterns, propose them for `code-base/`.

## Quality Standards
- Every TEMC score must be backed by specific data
- Every project must include a counter-argument section
- Stars > 500 threshold
- Active commits within last 90 days
- Permissive license (MIT/Apache/BSD)
